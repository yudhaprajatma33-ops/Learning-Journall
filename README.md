# Learning Journal - Reverse Engineering

## **Gambaran Umum**

Repository ini digunakan sebagai media dokumentasi pembelajaran selama mempelajari Reverse Engineering. 
Seluruh catatan yang tersimpan di dalamnya berisi rangkuman materi, hasil praktikum, serta refleksi pemahaman yang diperoleh pada setiap pertemuan. 
Dokumentasi ini dibuat untuk membantu proses belajar sekaligus menjadi arsip perkembangan kemampuan analisis perangkat lunak.

## **Tujuan Pembelajaran**

- Mendokumentasikan proses belajar Reverse Engineering secara berkala.
- Memahami konsep dan teknik analisis perangkat lunak.
- Meningkatkan kemampuan dalam mengidentifikasi struktur serta alur kerja program.
- Mengembangkan keterampilan penggunaan tools Reverse Engineering.
- Menjadi bahan evaluasi terhadap perkembangan pemahaman selama perkuliahan.

## **Isi Repository**

- Catatan pembelajaran mingguan selama satu semester.
- Ringkasan materi yang dipelajari pada setiap pertemuan.
- Dokumentasi hasil praktikum dan percobaan.
- Pembahasan tools yang digunakan dalam proses analisis.
- Refleksi mengenai pemahaman, kendala, dan solusi yang ditemukan.
- Rekam jejak perkembangan kemampuan dalam memahami executable dan binary program.
